# Flix GPT – AI-Powered Movie Discovery App

Flix GPT is a full-stack movie discovery platform that enables users to explore, search, and discover movies through AI-driven recommendations.  
It is built from the ground up using React, Tailwind CSS, Express, MongoDB, and AI APIs, with deployment on Firebase (frontend) and Render (backend).

---

## Features

- **AI-Driven Search** – Intelligent movie search powered by AI APIs.  
- **Modern Interface** – Fully responsive UI built with Tailwind CSS.  
- **User Authentication** – Secure signup and login using cookies and JWT.  
- **TMDB Proxy Server** – Custom Express.js proxy to bypass direct API restrictions.  
- **Backend with MongoDB** – MongoDB Atlas and Mongoose for user data and session management.  
- **User Notifications** – Real-time alerts using React Hot Toast.  
- **Deployment** – Frontend hosted on Firebase; backend deployed on Render.

---

## Tech Stack

**Frontend:** React.js, Tailwind CSS, React Hot Toast  
**Backend:** Node.js, Express.js, MongoDB Atlas, Mongoose  
**APIs:** TMDB API (via custom proxy), AI API  
**Deployment:** Firebase Hosting (frontend), Render (backend)
